# 📌 priority

| | |
| --- | --- |
| **Type** | select |
| **Source** | private `header/` **`priority`** field |
| **Allowed** | `1` · `2` · `3` · `4` · `5` |

Urgency when **Due Date** is soft — use with board sort (see [board-ui.md](../board-ui.md)).

| Rank | Meaning |
| ---: | --- |
| **1** | Legal, financial, pay, identity, career-critical |
| **2** | Health appointments, exams, nutrition cadence |
| **3** | Hygiene, grooming, home reset |
| **4** | Restock / buy lanes |
| **5** | Sync, media, low-stakes maintenance |

Personal tier examples: [engineering task-board](https://github.com/gardusig/engineering/blob/main/software-configuration/task-board.md).

**Upstream:** [Properties](./README.md)
